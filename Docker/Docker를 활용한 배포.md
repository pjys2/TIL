# Docker를 활용한 배포

## Docker 설치
1. Update the apt package index and install packages to allow apt to use a repository over HTTPS:
```
sudo apt-get update
```
```
sudo apt-get install \
    ca-certificates \
    curl \
    gnupg \
    lsb-release
```



[우분투 도커 설치 공식문서](https://docs.docker.com/engine/install/ubuntu/)
